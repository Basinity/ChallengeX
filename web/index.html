BUILD A CHALLENGE Try a starter challenge

BUILD HERE, PLAY THERE

01 COMPOSE

Pair triggers with effects, pick a goal, stack modifiers. Doubles, contradictions and terrible ideas are all allowed. The builder only checks the paperwork.

→
02 EXPORT

Download the preset as JSON, or copy a share link. The whole challenge lives in the URL. Post it in a video description, a Discord, anywhere.

→
03 IMPORT

Drop the file into config/challengex/presets/ and run /challengex import. Works in singleplayer and on servers.

GET THE MOD

ChallengeX is a server-side Fabric mod for Minecraft 26.2. Vanilla clients can join a server running it, and the same build covers singleplayer. This site only builds presets. The mod runs them.
